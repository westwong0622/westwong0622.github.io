如何更新主页

2026/02/06 posted in  Life Work
  • 想更新下自己的主页，让新朋友更快认识自己
  • 找工作的时候，也能更高效地展示自己

GitHub

建立 GitHub 主页

  • 建立一个和自己的 GitHub 账号同名的仓库即可
  • 比如 https://github.com/westwong0622/westwong0622

借鉴他人 GitHub 主页

  • 比如 https://github.com/steipete/steipete
    • 然后按需编辑即可，本质就是 markdown 而已
  • 或者你可以去看看更多人的
    • https://github.com/topics/github-profile-readme
  • 或者你也可以用 GUI 工具，帮你生成
    • https://github.com/rahuldkjain/github-profile-readme-generator

徽章小工具

  • https://shields.io 用来生成美观、统一、易读的徽章 badges 图像
    • 通常展示一些关键信息，例如：
      • 项目构建状态（passing / failing）
      • 包版本号
      • 代码覆盖率
      • 下载次数
      • 许可协议
      • 依赖情况
      • 贡献者数量
  • 比如
![Figma](https://img.shields.io/badge/-Figma-F24E1E?style=flat-square&logo=figma&logoColor=white)

  • 更多语法
    • 参考别人的就行，修改链接里的部分
      • 或者查看 https://shields.io/badges 学习下
    • Figma 是文本
    • 后面的是颜色值，可以用工具找到，见下
    • style 风格可以改为 flat, flat-square, plastic, for-the-badge, social
    • logo 需要你去搜索，见下
      • https://simpleicons.org 这个网站可以搜索图标，并复制这个品牌图标对应的颜色值，注意删掉 # 符号哦
  • 看我改完后的图标
![TypeScript](https://img.shields.io/badge/-TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)

查找 emoji

  • 在 readme 中可能会用到 emoji，你可以用 https://emojipedia.org 来查找
    • 输入法好烂啊，你懂的
  • 不止 GitHub 里的 markdown 会用到 emoji，凡是有文本的地方，你都可以尝试用
    • 注意不要太多哦，不然会妨碍阅读

Bonjour

  • 打开微信，扫码打开 Bonjour 小程序
    • 刚上来可能要输一些基本信息
  • 在我页面点击编辑主页
    • 修改状态：比如你想打 hackathon
      • 可以找人，bojour 会给你推荐哦
    • 修改个人简介：一段纯文本
    • 可以添加手机号、微信二维码图片：这个真的不错！
    • 修改一些 tag：你的职业、技能、所在城市、MBTI 等等
    • 往下就是 bento 形式的 block 了，你可以自由祝贺
      • 可以添加标题、文本
      • 可以添加网址（你做的项目），app store 链接、公众号也行
      • 可以添加图片（各种图片，你的自拍都行）
      • 添加一些职业经历、人生经历啥的
      • 最后就是你的 connect 方式了
        • 种类很多！！！
        • 社交媒体、开发者平台、设计平台、图文视频创作平台
        • 竟然还可以添加捐赠方式。。。
  • 看看我的 bonjour
