记录第一次直播

2026/02/06 posted in  Life Work
  • 今天尝试第一次直播，并写下笔记
  • 用的哔哩哔哩，其它的渠道以后再说

哔哩哔哩

  • 背景：设备为 macOS

基本步骤

  • 官网下载 b 站 macOS 客户端 https://app.bilibili.com
    • 打开软件并登录
  • 首页切换到直播模块，点击右下角直播按钮
    • 选择分享内容：类似于视频会议，可以选择全屏或某个窗口
    • 点击开始直播按钮
      • 如果没有实名认证，会提醒实名认证
      • 用手机客户端扫码认证，认证很快的
    • 点击分享直播间，测试一下
      • 可以复制链接
      • 也可以用微信扫码查看

分享音频

  • 如何分享其它软件声音，比如放点音乐
    • 到 BlackHole 官网 https://existential.audio/blackhole 填写邮箱，可免费下载
    • 安装 BlackHole 并重启电脑（是驱动级别的软件！）
    • 重新开直播，并添加素材
      • 选择音频设备
      • 选择 BlackHole 虚拟音频
    • 打开 Control Center 并切换系统声音输出为 BlackHole
      • 测试下直播链接能不能听到你放的音乐
      • 但是你的耳机就听不到了，哈哈
  • 如何既可以把内置声音通过 BlackHole 播出去，同时可以自己听到？
    • 通过 Spotlight 打开音频 MIDI 设置
      • 在左下角点 ＋ 号并选择 创建多输出设备
      • 在右侧勾选
        • ✅ BlackHole
        • ✅ 你的耳机（USB / 蓝牙 / 有线都行
      • 注意以下设置
        • 主设备：选你的耳机（防止延迟）
        • 漂移校正：只给 BlackHole 勾上 ✅
    • 打开 Control Center 并切换系统声音输出为 BlackHole 新建的「多输出设备」
      • 测试下能不能同时听到

额外知识点

  • 下载 BlackHole 有 2ch、16ch 可以选，选什么？
  • BlackHole 2ch 是什么
    • 2ch = 双声道（左 / 右）的特点
      • 🎧 和普通耳机 / 音箱一样
      • 🎵 Apple Music、网页音乐、视频，全是 2ch
      • ⚙️ 配置最简单
      • 🧠 几乎不可能配错
  • BlackHole 16ch 是什么
    • 16ch = 16 路独立音频通道
    • 你可以拿它干嘛？举个真实场景 👇
通道 用途
ch1–2 Apple Music
ch3–4 游戏声音
ch5–6 浏览器
ch7–8 提示音
ch9–16 备用
  • 用一个类比你秒懂
    • 2ch = 一个水龙头：打开就有水，怎么都不会错
    • 16ch = 一个 16 路水阀面板：很强，但你要知道每根管子接哪
  • 默认选 2ch 就行
